IOC Report

Author : https://github.com/leunammejii
Project: https://github.com/leunammejii/ioc_report

Description

This site is used as a way to request the IOCs unique to your environment where the initial indicator was a URL hosting a malicious file. This will allow you to correlate data with other log sources in order to better understand the threat.

Request All IOCs

Sending a GET request to /csv will return all of the IOCs in CSV format listed by URL. Or you can get all of the IOCs as a flat text file listed by URL by send a GET request to /text.

Request IOCs for a given MD5

Sending a GET request to /{FORMAT}/{MD5} will return the IOCs in requested format for that MD5. Here is an example:

http://{{ sena }}/csv/{{ chsu }}
