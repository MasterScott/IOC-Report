IOC Report

Author : https://github.com/leunammejii
Project: https://github.com/leunammejii/ioc_report

Description

This site is used as a way to request the IOCs unique to your environment where the initial indicator was a URL hosting a malicious file. This will allow you to correlate data with other log sources in order to better understand the threat.

Request All IOCs

Return all of the IOCs in CSV format listed by URL -> GET http://{{ sena }}/csv
Return up to 1000 of the last report summaries from the Hybrid Analysis feed in CSV format -> GET http://{{ sena }}/feed
Return all of the IOCs as a flat text file listed by URL -> GEThttp://{{ sena }}/text

Request IOCs for a given IOC

Sending a GET request to /{FORMAT}/{IOC} will return data in requested format for that IOC. Here is an example:

http://{{ sena }}/csv/{{ chsu }}
